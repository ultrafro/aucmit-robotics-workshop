# AUC × MIT Robotics Workshop

Website for a student-led educational partnership between The American University in
Cairo and MIT: MIT students travel to AUC to teach a hands-on robotics workshop built
on [LeRobot](https://github.com/huggingface/lerobot), Orange Pi 5 boards, and stereo
camera vision.

Static site (plain HTML/CSS in `public/`), deployed on Vercel.

- `public/index.html` — landing page
- `public/bom.html` — bill of materials (linked from the footer at `/bom`)
